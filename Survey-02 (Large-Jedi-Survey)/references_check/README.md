# Проверка источников на Google Scholar

## 📋 Описание

Скрипт использует библиотеку `scholarly` для автоматического поиска статей на Google Scholar и проверки наличия PDF версий.

## 🚀 Установка

### 1. Установите зависимости:

```bash
pip install scholarly
```

### 2. Запустите скрипт:

```bash
cd C:\xampp\htdocs\Online-Surveys\LargeJediSurvey
python check_references.py
```

## 📊 Что делает скрипт

1. **Поиск статей** на Google Scholar по названию
2. **Проверка наличия PDF** (url_pdf или url_eprint)
3. **Сохранение результатов**:
   - `references_check/results_*.json` — полные данные
   - `references_check/summary_*.txt` — краткая сводка

## 🔍 Преимущества scholarly

- ✅ **Официальный API** Google Scholar
- ✅ **Не требует парсинга HTML**
- ✅ **Стабильные результаты**
- ✅ **Извлекает метаданные** (цитирования, авторы, год)
- ✅ **Находит PDF ссылки** (если доступны)

## ⚠️ Ограничения

- Google Scholar может **блокировать частые запросы**
- Некоторые статьи **не имеют открытых PDF**
- Книги редко доступны бесплатно

## 💡 Рекомендации

1. **Не запускайте слишком часто** — делайте паузы между запусками
2. **Проверяйте вручную** источники без PDF
3. **Используйте DOI** для поиска на сайтах издательств
4. **Для книг** проверяйте сайты издательств напрямую

## 📞 При проблемах

Если scholarly не работает:
1. Проверьте интернет-соединение
2. Попробуйте использовать прокси/VPN
3. Увеличьте REQUEST_DELAY в скрипте
4. Проверьте, не заблокирован ли доступ к Google Scholar
