# Проверка русскоязычных источников на нескольких ресурсах

## 📋 Описание

Скрипт проверяет наличие русскоязычных источников на 4 ресурсах:
- **CyberLeninka.ru** — открытая научная библиотека
- **eLibrary.ru** — крупнейшая российская научная библиотека
- **Google Scholar** — международная база публикаций
- **PsyJournals.ru** — психологические журналы (проверяется автоматически через API)

## 🚀 Установка

1. Установите Python 3.8+ (если ещё не установлен)

2. Установите зависимости:
```bash
pip install -r requirements_check.txt
```

## 📖 Использование

### Быстрый запуск:
```bash
python check_references.py
```

### Результаты:
- `references_check/results_YYYY-MM-DD_HH-MM-SS.json` — полные результаты (JSON)
- `references_check/summary_YYYY-MM-DD_HH-MM-SS.txt` — краткая сводка (TXT)

## ⏱️ Тайминги

Константы настраиваются в начале скрипта `check_references.py`:

```python
REQUEST_DELAY = 20        # Пауза между запросами в секундах
ERROR_429_DELAY = 60      # Пауза при ошибке 429 в секундах
MAX_RETRIES = 3           # Максимальное количество попыток при ошибке 429
```

**Время проверки 10 источников:** ~10-15 секунд (только генерация ссылок)

PsyJournals проверяется автоматически через API. Остальные ресурсы генерируют ссылки для ручной проверки.

## 📊 Интерпретация результатов

| Статус | Значение | Что делать |
|--------|----------|------------|
| ✅ FOUND | Источник найден автоматически (PsyJournals API) | Оставить в документе |
| ❌ NOT_FOUND | Требуется ручная проверка | Открыть ссылки в браузере |

## 🔍 Добавление новых источников

Откройте `check_references.py` и добавьте в список `SOURCES_TO_CHECK`:

```python
{
    'id': 'unique_id',
    'query': 'поисковый запрос',
    'expected': 'ожидаемое название работы',
    'hypothesis': 'H1, H2'  # какие гипотезы используют
},
```

## 🌐 Альтернативные базы

Если источник не найден, проверьте:

1. **eLibrary.ru** — https://www.elibrary.ru (требуется регистрация)
2. **Google Scholar** — https://scholar.google.ru
3. **ResearchGate** — https://www.researchgate.net
4. **PsyJournals.ru** — https://psyjournals.ru
5. **Сайты издательств** — Питер, Юрайт, КноРус, и т.д.

## ⚠️ Ограничения

- **CyberLeninka, eLibrary, Google Scholar** — только генерация ссылок (требуется ручная проверка)
- **PsyJournals** — проверяется автоматически через API
- Книги (монографии) редко представлены в открытом доступе

## 📞 Контакты

При проблемах со скриптом:
1. Проверьте подключение к интернету
2. Откройте ссылки вручную в браузере
3. Для книг проверьте сайты издательств
