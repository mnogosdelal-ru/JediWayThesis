# Инструкция по тестированию

## 📋 План тестирования

### Этап 1: Базовое тестирование

#### 1.1 Проверка установки

```bash
# 1. Проверьте, что MySQL запущен
mysql -u root -p -e "SHOW DATABASES;"

# 2. Проверьте базу данных
mysql -u root -p -e "USE jedi_survey; SHOW TABLES;"

# 3. Проверьте, что таблицы созданы
mysql -u root -p -e "USE jedi_survey; DESCRIBE respondents;"
```

**Ожидаемый результат:**
- База данных `jedi_survey` существует
- Таблицы `respondents` и `aggregates` созданы
- Структура таблиц соответствует миграции

#### 1.2 Проверка конфигурации

1. Откройте `config/config.php`
2. Проверьте настройки БД:
   - `DB_HOST` = `localhost`
   - `DB_NAME` = `jedi_survey`
   - `DB_USER` = `root`
   - `DB_PASS` = (ваш пароль)

#### 1.3 Первый запуск

1. Откройте браузер
2. Перейдите на: `http://localhost/Online-Surveys/LargeJediSurvey/public/`
3. Должна появиться страница "Информированное согласие"

**Ожидаемый результат:**
- ✅ Страница загрузилась
- ✅ Прогресс бар показывает 0%
- ✅ Кнопка "Продолжить" активна

1. Скопируйте код респондента со страницы 10
2. Откройте: `http://localhost/Online-Surveys/LargeJediSurvey/public/api.php?code=ABC123`

**Ожидаемый результат:**
```json

  "success": true,
  "respondent": {

    "code": "ABC123",

  "scores": {
    "mijs_total": 34,
```

## 🐛 Отладка

### Проблема: Ошибка подключения к БД

**Решение:**
1. Проверьте, что MySQL запущен
2. Проверьте логин/пароль в `config/config.php`
3. Проверьте, что БД существует

### Проблема: Страница не загружается

**Решение:**
1. Проверьте логи Apache/PHP
2. Проверьте права доступа к папкам
3. Включите `DEBUG_MODE` в `config.php`

### Проблема: AJAX не работает

**Решение:**
1. Откройте консоль браузера (F12)
2. Проверьте ошибки JavaScript
3. Проверьте Network tab — отправляется ли запрос на `save.php`
