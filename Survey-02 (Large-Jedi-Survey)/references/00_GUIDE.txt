1. ZOTERO (бесплатно, рекомендуется):

      - Выберите references.bib
      - Все 34 работы импортируются автоматически

   а) Импорт из BibTeX:
      - File → Import → BibTeX
      - Выберите references.bib

      - Tools → Import from DOI
      - Вставьте DOI из ALL_DOIs.txt

3. ENDNOTE:
   
   - File → Import → File
   - Выберите references.bib
   - Import Option: Choose → BibTeX

ПРИОРИТЕТ 1: Open Access (бесплатно)
----------------------------------------
Следующие работы должны быть доступны бесплатно:

1. Sirois & Pychyl (2013) - Open Access
   https://doi.org/10.1111/spc3.12011

2. Norman (2010) - Springer Open Choice
   https://doi.org/10.1007/s10459-010-9222-y

3. Cohen & Janicki-Deverts (2012) - PMC
   https://www.ncbi.nlm.nih.gov/pmc/articles/PMC3589142/


ПРИОРИТЕТ 2: Institutional Access
----------------------------------------
Если у вас есть университетская подписка:

1. Зайдите на сайт библиотеки вашего вуза
2. Найдите доступ к базам:
   - APA PsycNet (для статей APA)
   - ScienceDirect (для Elsevier)
   - JSTOR (для старых статей)
   - SpringerLink (для Springer)
3. Введите DOI в поиск на сайте библиотеки


ПРИОРИТЕТ 3: ResearchGate / Авторские страницы
----------------------------------------
Многие авторы выкладывают препринты:

1. https://www.researchgate.net/
   - Введите название статьи в поиск
   - Нажмите "Request full-text" если нет доступа

2. Google Scholar:
   - Введите: [Author] [Year] [Title] PDF
   - Ищите ссылки с пометкой [PDF]


ПРИОРИТЕТ 4: Unpaywall / Open Access Button
----------------------------------------
Легальные сервисы для поиска OA версий:

1. Unpaywall: https://unpaywall.org/
   - Установите браузерное расширение
   - При заходе на статью ищите зелёный замок

2. Open Access Button: https://openaccessbutton.org/
   - Введите DOI или URL
   - Найдёт бесплатную версию если есть


ПРИОРИТЕТ 5: Межбиблиотечный абонемент (МБА)
----------------------------------------
Если статья недоступна:

1. Обратитесь в библиотеку вашего вуза
2. Запросите статью через МБА
3. Обычно занимает 3-10 дней
4. Часто бесплатно для студентов/аспирантов


ПРИОРИТЕТ 6: Написать автору
----------------------------------------
Авторы могут легально делиться своими работами:

1. Найдите email corresponding author
2. Напишите вежливое письмо:
   
   "Уважаемый [Имя],
   
   Я аспирант/исследователь, работаю над исследованием 
   продуктивности. Ваша статья [название] очень важна 
   для моей работы. Не могли бы Вы поделиться PDF?
   
   С уважением,
   [Ваше имя]"

3. Авторы обычно отвечают в течение 1-2 недель
