# KODA.md — Инструкции для работы с проектом Jedi Boxes

## Обзор проекта

**Название:** Jedi Boxes (Три коробочки)  
**Тип:** Веб-приложение для проведения психологического исследования  
**Регистрация:** OSF.IO/NA62T  
**Цель:** Валидация инструмента оценки личной продуктивности через распределение 6 единиц энергии по трём зонам

### Технологический стек

- **Frontend:** HTML5, JavaScript (ES6+), CSS3
- **Backend:** PHP 7.4+ (чистый PHP, без фреймворка)
- **База данных:** MySQL 8.0+
- **Веб-сервер:** Apache/Nginx с поддержкой PHP

### Структура проекта

**Корневая директория:** тестовые файлы прототипирования (можно игнорировать)

```
Survey-Jedi-Boxes-Validation/     # Основная директория приложения
    ├── index.html                # Главная страница опроса
    ├── result.html               # Страница результатов респондента
    ├── app.js                    # Логика приложения (клиент)
    ├── api.php                   # Серверный API
    ├── drag-drop.js              # Логика drag-and-drop кубиков
    ├── style.css                 # Стили приложения
    ├── db_config.php             # Конфигурация подключения к БД
    ├── monitor.php               # Страница мониторинга (админ)
    ├── setup_db.sql              # Скрипт создания БД
    └── respondents/              # JSON-файлы с ответами респондентов
```

## Основные компоненты

### 1. Опрос (index.html + app.js)

Респондент проходит 6 страниц:
- **Страница 0:** Согласие + демография (возраст, пол, должность, профессия)
- **Страница 1:** Распределение 6 кубиков по зонам + вопрос про память/записи
- **Страница 2:** Контекстные вопросы (типичность недели, баланс работа/личное, дефицит энергии)
- **Страница 3:** Шкала прокрастинации (8 вопросов)
- **Страница 4:** Шкала удовлетворённости жизнью SWLS (5 вопросов)
- **Страница 5:** Шкала эмоционального истощения MBI (9 вопросов)
- **Страница 6:** Завершение + редирект на результаты

## Развёртывание и запуск

### Требования

- PHP 7.4 или выше
- MySQL 8.0 или выше
- Веб-сервер (Apache/Nginx)

### Установка

1. **Настройка базы данных:**
   ```bash
   mysql -u root -p < Survey-Jedi-Boxes-Validation/setup_db.sql
   ```

2. **Настройка подключения к БД:**
   Отредактируйте `Survey-Jedi-Boxes-Validation/db_config.php`:
   ```php
   define('DB_HOST', 'localhost');
   define('DB_NAME', 'jedi_way');
   define('DB_USER', 'ваш_пользователь');
   define('DB_PASS', 'ваш_пароль');
   ```

3. **Настройка веб-сервера:**
   - Направьте Document Root на папку с `index.html`
   - Убедитесь, что включен mod_rewrite (если используется)
   - Настройте обработку PHP файлов

4. **Права доступа:**
   - Папка `respondents/` должна быть доступна для записи (для сохранения JSON)
   - Остальные файлы — только для чтения

### Запуск

После настройки откройте `index.html` в браузере. Опрос работает как одностраничное приложение (SPA).

## Известные особенности и ограничения

### Индикаторы на странице результатов

Для трёх шкал отображаются цветные эмодзи:
- **Прокрастинация:** ≤15 🟢, 16-22 🟡, ≥23 🔴
- **Удовлетворённость (SWLS):** ≤16 🔴, 17-24 🟡, ≥25 🟢
- **Эмоциональное истощение (MBI):** ≤15 🟢, 16-24 🟡, ≥25 🔴

### Метаданные

Поля `user_agent`, `ip_hash`, `device_type` должны заполняться автоматически при создании сессии. Если они пустые — проверьте:
1. Выполнен ли `setup_db.sql` на боевом сервере
2. Нет ли ошибок в консоли браузера при init_session

### Отладка

Добавьте `?debug=true` к URL для включения режима отладки:
```
index.html?debug=true
```
